# Excel Sales Dashboard

## Project Overview
This project is an interactive Excel Sales Dashboard created using Pivot Tables, Pivot Charts, Slicers, and KPI Cards.  
It helps analyze sales performance across different categories, regions, and time periods.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features
- Total Sales Overview
- Profit Analysis
- Category-wise Sales
- Region-wise Performance
- Monthly Sales Trend
- Interactive Slicers for filtering

## Key Insights
- Identified top-performing categories
- Compared regional sales performance
- Tracked monthly sales trends
- Improved business decision-making using data visualization
